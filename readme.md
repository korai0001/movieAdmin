動画管理システム  
  
① ソフトの目的  
作者である紅雷が、ローカルサーバー内の動画ファイルをフォルダ単位で管理していたのだが、  
量が多くなりすぎて参照しにくくなったので、参照しやすいように作ったローカル用の補助ツールです。  
htaで書かれているので改造は極めて容易という利点があります  
  
② ライセンス  
このアプリケーションに含まれるソース一式は、  
すべてGPL v2.0のライセンスにおいて再頒布を許可することとします。  
GPL v2.0に関する詳しい情報は

で確認してください。  
また、将来このアプリケーションがGPL v3.0以降に  
ライセンスを移行することはありません。  
  
③設定方法  
config.vbsにフォルダのルートディレクトリを記載してください  
以下のフォルダが、下記のようになっている想定です  
１階層目  
・「あ～わ」までの固定フォルダ  
２階層目  
・各作品名フォルダ  
３階層目  
・各動画ファイル
